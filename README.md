# Esp-with-LoRa-and-WiFi
In this project Esp32 and LoRa module is used to receive data from other LoRa node and to upload that data to the server of your own.

# Pin-Connections

3.3V: 3.3V

GND: don’t connect

RESET: GPIO 14
NSS: GPIO 5

MOSI: GPIO 23
